# เครื่องพิมพ์สามมิติ Bear SKR V1
## บทนำ
เครื่องพิมพ์สามมิติ Bear SKR เป็นเครื่องพิมพ์สามมิติสำหรับผู้ที่สนใจสร้างเครื่องพิมพ์สามมิติที่มีโครงสร้างแข็งแรง เชื่อถือได้ ทำให้พิมพ์งานได้เร็วขึ้น มีความแม่นยำสูง และซ่อมบำรุงง่าย
